## a DSL for Speedsolvers with a JavaScript Interpreter

### Syntax:
- State (...)
- Only (...)
- Moves (...)
- State ( Scrambled )


### Example

```
generate different Ua permutations
State (
    rrr
    rrr
    rrr
gyg yby bgb www
ggg yyy bbb www
ggg yyy bbb www
    ooo
    ooo
    ooo
)

Only (
   U U'
   R R'
)

Moves (15)

State (
    rrr
    rrr
    rrr
ggg yyy bbb www
ggg yyy bbb www
ggg yyy bbb www
    ooo
    ooo
    ooo
)
```
